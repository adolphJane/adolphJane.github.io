编译OpenWrt系统

2022/07/29 posted in  工具

OpenWrt官网地址
Lean版OpenWrt
常用插件库地址

目前Lean版OpenWrt比较流行，也支持很多插件扩展，本文选用Lean版OpenWrt进行编译

注意
不要用 root 用户进行编译
国内用户编译前最好准备好梯子
默认登陆IP 192.168.1.1 密码 password

编译

编译系统

首先装好 Linux 系统，推荐 Debian 11 或 Ubuntu LTS

安装编译依赖

更新apt

sudo apt update -y

升级apt

sudo apt full-upgrade -y

安装依赖

sudo apt install -y ack antlr3 asciidoc autoconf automake autopoint binutils bison build-essential \
bzip2 ccache cmake cpio curl device-tree-compiler fastjar flex gawk gettext gcc-multilib g++-multilib \
git gperf haveged help2man intltool libc6-dev-i386 libelf-dev libglib2.0-dev libgmp3-dev libltdl-dev \
libmpc-dev libmpfr-dev libncurses5-dev libncursesw5-dev libreadline-dev libssl-dev libtool lrzsz \
mkisofs msmtp nano ninja-build p7zip p7zip-full patch pkgconf python2.7 python3 python3-pip libpython3-dev qemu-utils \
rsync scons squashfs-tools subversion swig texinfo uglifyjs upx-ucl unzip vim wget xmlto xxd zlib1g-dev

下载源代码

拉取OpenWrt源码

git clone https://github.com/coolsnowwolf/lede

安装插件

切换目录

cd lede

1.配置插件仓库，拉取插件

配置插件仓库地址

sed -i '$a src-git kenzo https://github.com/kenzok8/openwrt-packages' feeds.conf.default

sed -i '$a src-git small https://github.com/kenzok8/small' feeds.conf.default

拉取插件

git pull

2.单独拉取插件代码

定位OpenWrt插件目录

cd package/lean

git clone ****[插件地址]*****

更新 feeds 并选择配置

更新Feed

./scripts/feeds update -a
./scripts/feeds install -a

安装插件

编译

make menuconfig

编译完成，会显示如下界面

注意，需要把终端大小调整到至少19行80列，不然会报错，无法进入到该设置界面。
Target Images 里面把GZip images按Y勾上，编译出来的固件是压缩包格式，会比较小，下载也比较快

下载dl库，编译固件

下载DL库

make download -j8

不能用root用户编译
-j 后面是线程数，第一次编译推荐用单线程

make V=s -j1

编译完成

编译完成后输出路径：bin/targets
