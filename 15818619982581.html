IDEA 远程一键部署Spring Boot到Docker

2020/03/25 posted in  JavaWeb

CentOs安装Docker

移除旧版本

yum remove docker \
                  docker-client \
                  docker-client-latest \
                  docker-common \
                  docker-latest \
                  docker-latest-logrotate \
                  docker-logrotate \
                  docker-selinux \
                  docker-engine-selinux \
                  docker-engine

安装一些必要的系统工具

yum install -y yum-utils device-mapper-persistent-data lvm2

添加软件源信息

yum-config-manager --add-repo http://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo

更新 yum 缓存

yum makecache fast

安装 Docker-ce

yum -y install docker-ce

如果报package docker-ce-3:19.03.11-3.el7.x86_64 requires containerd.io >= 1.2.2-3, but none of the providers can be installed错误

进入阿里云镜像地址：https://mirrors.aliyun.com/docker-ce/linux/centos/7/x86_64/edge/Packages/找到你想要的或者最新的containerd.io包，拼接在阿里云地址后面。
可以执行下方的：

yum install -y https://mirrors.aliyun.com/docker-ce/linux/centos/7/x86_64/edge/Packages/containerd.io-1.2.13-3.1.el7.x86_64.rpm

或者自拟

接着运行sudo yum install docker-ce

启动 Docker 后台服务

systemctl start docker

镜像加速

vim /etc/docker/daemon.json

在该配置文件中加入（没有该文件的话，请先建一个)

{
  "registry-mirrors": ["http://hub-mirror.c.163.com"]
}

测试能否使用，运行hello-world

docker run hello-world

配置docker远程连接端口

 vi /usr/lib/systemd/system/docker.service

找到 ExecStart， 在ExecStart=/usr/bin/dockerd之后添加 -H tcp://0.0.0.0:2375，如下图所示

为了安全，推荐使用2376端口

2375端口会被人恶意扫描入侵

TLS配置查看Centos7 Docker配置TLS认证的远程端口的证书生成教程（shell脚本一键生成）

重新加载Docker配置

systemctl daemon-reload // 1，加载docker守护线程
systemctl restart docker // 2，重启docker

查看Docker状态

systemctl status docker
