转变心态，营销原则

2021/11/07 posted in  营销

学会营销自己

  1. 找工作，销售出去
  2. 找女朋友，销售出去
  3. 老说缺钱，不去找钱，重要环节

营销模式

  1. 等钱找你， 工作 -> 死工资 -> 没有规划
  2. 工作同时，整合资源
    1. 有技术能力
    2. 做事，交流，讨论，表达出我能做事
    3. 有事的时候才会找你
  3. 要敢于做事，不要老想回报
  4. 认真的规划做事
    1. 跟着有前途的公司成长，赚到第一桶金，积累人脉、资源
  5. 穷看书(提升自己)，富交友

财富自由的心态

  1. 95%的人无法实现财务自由
  2. 人生每个阶段的目标都是不同的
  3. 财富
    1. 身体健康，家人平安
    2. 基本的生活保障(车，房，教育-学会知足)(物质)
    3. 做一些正向的有价值，有意思的事情(精神)
      1. 赚多少钱没关系
      2. 这个事情本身到底有多少正向价值
      3. 做这个事情赚到的钱，其实并不是你的，而是社会的
      4. 提升个人的社会地位
      5. 能坚持下来的，都是喜欢做事的
    4. 你自己的能力，你的人事，帮助家族，精神财富

没有原始资本怎么办

  1. 没有规划好自己，长期的规划，短视 —— 几年过去了，没有什么积累
  2. 定力的去奋斗
  3. 给你500W，你多久可以给我回报一个亿，你的计划是什么，你凭什么实现(做不成不在原始资本)
    1. 你在一个赚钱行业
    2. 你在这个行业里面积累，赚钱了
    3. 你想复制一个，趁着时间差
    4. 失败的案例，失败也不重要，看得清，做事过程中 —— 认识到新事物
  4. 人不应该小富即安，偏安一隅

做产品

  1. 外包
    1. 准备技术
    2. 准备产品、框架、架构师
    3. 有市场主流的Demo
    4. 花费一段时间，完成积累
  2. 如何让别人相信
    1. 自由要有本事 -> 商业价值产品
    2. 把产品销售出去
    3. 总可以找到本事比你厉害，境遇和你一样的人
    4. 为什么有人能赚到钱，有人赚不到，做个提前量打时间差(抖音，短视频)
    5. 为此要提前做好规划和个人技术提升

卖东西，面子

  1. 不要想是赚别人钱的事
  2. 需要提供相应的价值
  3. 是理所当然，不要过意不去
  4. 方式
    1. open boom， 朋友
    2. 亲兄弟，明算帐，讲规则不讲情谊
  5. 能通过商业的不走人情

学会让别人赚你的钱

  1. 让钱来调度资源
  2. 学会合理分配资源

做人做事，要有底线

  1. 梳子不要卖给和尚
  2. 守住底线
  3. 要有核心竞争力
  4. 敢承诺，敢说

商业本质

  • 做事，提供产品服务型
    1. 盈利模式
    2. 获客， ToC， ToB
      1. 实体店： 周边人流引入
      2. 放海报，做活动，搞促销
      3. 广告牌
    3. 销售转化
      1. 如何卖出你的东西
    4. 产品
      1. 进货渠道
      2. 供应链
    5. 运营
      1. 让用户在你这里觉得有趣，复购，介绍
    6. 管理
      1. 数据管理
      2. 团队管理
  • 做平台型
    1. 抓住用户
      1. 通用用户
      2. 垂直领域
    2. 让内容在平台上赚到钱
    3. 基于平台来做商业模式
      1. 流量变现
      2. 资金支持

创业打法

  1. 做小
    1. 客户获得产品，产生价值
    2. 也可以为做大做准备
    3. 是否有长期的价值
    4. 能看到天花板
    5. 要有现金流
  2. 做大
    1. 做平台

总结

  1. 大家生活都是一样的，没有本质的区别
  2. 读了大学，基本都能很好的实现生活
  3. 做好长期规划 —— 有前景的行业+公司
  4. 多交流，站在做事的角度，而不是工作的角度，跨界交流
  5. 做事情不要强求启动资金，开始做+做的过程中寻找机会，只要你在做，就有机会拿到资源，不要整天想着赚小钱，没有小富即安，只有不断的做事情
  6. 这些经历就是你的精神财富，总结的越透彻，对下一代帮助越多
  7. 要学会商业
    1. 赚别人的钱
    2. 学会花钱
  8. 商业活动中要有底线，无论有多难

销售的基本流程

破冰/接待

  1. 交流会：介绍自己
    1. 我是谁？来自哪里？
    2. 我能给大家带来什么样的资源
  2. 准备几套话术
    1. 销售标准化流程

挖掘出来用户的需求

  1. 特定的需求
  2. 陌生的销售
    1. 快速了解它的需求，然后了解痛点

介绍我们的产品如何能解决问题

  1. 使用类
  2. 功能类

塑造一个价值，让别人愿意为你这个产品付费

  1. 销售理念
  2. 特惠
  3. 解决我们在销售环节中客户的疑问

熟悉你的产品（专业）

熟悉你的竞品

宣讲你的品牌

建立信任

  1. 人的信任
  2. 产品专业的信任

合格销售如何学会倾听

  1. 态度： 多听，慢点发表自己的意见，营造一个氛围，让别人多说
    1. 提示关键信息
    2. 讲述者逻辑完整的走完整个问题逻辑
    3. 对别人观点表示认同，在对方发表异议时，寻找反转观点
    4. 梳理逻辑并总结
  2. 不要着急给方案
  3. 得到确认以后给方案
  4. 争一个对错没有意义，解决问题才有意义

合格销售如何挖掘客户的痛点

  1. 客户遇到问题，表达的还不够痛，梳理痛点
    1. 提升认同感
  2. 客户遇到痛点，他自己可能没有意识到
    1. 提升专业感和认同感
  3. 描述场景方面，有感
  4. 通过沟通以及细节判断

如何把自己的产品卖出去

  1. 掌握客户的基本情况
  2. 帮助客户梳理他的需求点
  3. 介绍你的产品时如何解决的，效果如何
  4. 怎么证明你说的是真的
  5. 为什么要现在马上下单购买

如何逼单，销售常用技巧

  1. 想买又不买
  2. 再次挖掘客户深层次的疑虑到底是什么
    1. 技术上的疑虑，实实在在的疑虑
    2. 心理上的疑虑，不想冲动消费

如何带好销售团队与销售管理

  1. 产品： 设计一个完整的销售流程，植入品牌，特点、功能
  2. 培训我们的销售对应话术
    1. 破冰
    2. 目标价位、工作
    3. 介绍产品
    4. 价格与活动
  3. 转化率

销售项目

程序员谈外包

  1. 需求分析
    1. 倾听这个外包干嘛用的，这个人是干嘛的
    2. 你打算花多少钱做这个事情，外包的预算
    3. 整理一下整个的实现方式，大体的技术方案，遗漏，复杂的功能
    4. 帮助用户梳理出来，他要做的东西，如何落地，同时确认一下需求
    5. 定制开发
    6. 换皮
    7. 报价，时间
  2. 技术方案与工作量的确定（收不了尾，但是客户又要加功能，又不加钱）
    1. 做好项目原型设计，技术点的工作量
    2. 对方能提供什么
      1. 美术负责、策划负责
    3. 一个报价+时间
      1. 结合自己 - 兼职做，还是全职做（日薪 * 3）
      2. 结合对方 - 对方预算范围大概多少
      3. 他认知里面的行情是多少
    4. 方案=产品，专业+案例
    5. 买贵了，卖便宜
  3. 合同最终的责任，价格谈判
    1. 开案费（30% ～ 60%）
    2. 主体的功能完成（60%～80%）
    3. 完整的上线（20%）
    4. 维护时间段，相关协议
